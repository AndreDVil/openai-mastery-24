# Project 01 – Demo (Stateful CLI Chat Client)

This demo showcases the **stateful chat client** developed for Project 01.  
It demonstrates:

- Model configuration via CLI  
- Multi-turn conversation with memory  
- Internal commands (`/config`, `/history`, `/clear`, `/exit`)  
- Latency and token usage  
- Actual execution of the `basic_chat_stateful.py` script  

This transcript was generated by running the script directly in a terminal.

---

## 1. How to Run

Use:

```bash
python projects/p01_basic_chat/basic_chat_stateful.py \
  --model gpt-4o-mini \
  --temperature 0.7 \
  --top_p 1.0 \
  --max-tokens 128
```
